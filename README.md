# Nephropathology Self-Study Portal

Interactive self-assessment portal for nephropathology education.

## Features

- Bilingual support (English/Lithuanian)
- Multiple choice questions with explanations
- Image-based questions with zoom functionality
- Progress tracking
- Score calculation

## Access

Visit: https://arvylau.github.io/Nephro4Students/

## Usage

1. Select your preferred language (EN/LT)
2. Answer questions by clicking on options
3. View explanations after each answer
4. Track your progress and score
